🎬 Netflix-Data-Analysis-Project-Using-Python

---

📌 Project Overview 

 This project analyzes Netflix Movies and TV Shows data using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn. The goal is to explore the dataset, clean missing values, and generate insights through data analysis and visualization.

---

  🎯 Objective

The objective of this project is to analyze Netflix movies and TV shows data using Python libraries such as pandas, numpy, matplotlib, and seaborn. This project helps in understanding content trends, genres, ratings, countries, and release patterns available on Netflix.

---

 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
---

 📌 Key Insights

* Movies are more common than TV Shows on Netflix.
* The United States contributes the highest amount of content.
* Drama and Comedy are the most popular genres.
* Netflix content increased rapidly after 2015.
* TV-MA is one of the most frequent content ratings.

---

 🚀 Conclusion

This project demonstrates how Python libraries can be used for real-world data analysis and visualization. By analyzing Netflix data, we gained insights into content distribution, trends, and audience-targeted ratings.
