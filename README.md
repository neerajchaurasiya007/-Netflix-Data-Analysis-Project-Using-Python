🎬 Netflix-Data-Analysis-Project-Using-Python


📌 Project Overview 

 This project analyzes Netflix Movies and TV Shows data using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn. The goal is to explore the dataset, clean missing values, and generate insights through data analysis and visualization.


  🎯 Objective

The objective of this project is to analyze Netflix movies and TV shows data using Python libraries such as pandas, numpy, matplotlib, and seaborn. This project helps in understanding content trends, genres, ratings, countries, and release patterns available on Netflix.



 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
